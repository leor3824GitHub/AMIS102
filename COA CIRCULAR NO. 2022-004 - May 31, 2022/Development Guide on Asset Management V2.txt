This comprehensive Domain Entity Plan is designed for your Asset Management and Inventory System (AMIS). It follows a "Ledger-First" architecture, ensuring that every piece of government property—from a ₱100,000 industrial rice mill to a ₱3,000 office chair—maintains a perfect audit trail compliant with COA Circular 2022-004.
AMIS Development Guide: Unified Asset & Lifecycle Domain
1. Core Architectural Philosophy

The system treats Classification (PPE vs. Semi-Expendable) and Accountability (Custodian) as "point-in-time" states.

    Dynamic Thresholds: Instead of hardcoding IsPPE, the system uses a configurable policy. This allows for Reclassification when government thresholds change.

    Side-Effect Updates: Fields like CurrentCustodianId are never updated manually; they are automated side-effects of saving a Transaction (PAR, ICS, ITR, etc.).

This allows the system to change rules without a code redeploy.

3. Detailed Development Guide
I. Automated Custodian Logic

To maintain data integrity, the CurrentCustodianId on the GovernmentProperty table must be updated automatically within the Transaction Service:

    Issuance (PAR/ICS): Sets CurrentCustodianId to the ReceivedById.

    Transfer (PPEIR/SMIR): Updates CurrentCustodianId to the new ReceivedById.

    Return (RRSP): Resets CurrentCustodianId to null (returned to stock).

II. The Reclassification Workflow ("Affective PPE")

When a new policy is activated:

    Identify: Scan for assets where CurrentCategory == PPE but UnitCost < NewCapitalizationThreshold.

    Log: Create a ClassificationLog record.

    Re-Issue: Trigger a service to void the active PAR and generate a new ICS for the current custodian. This ensures the custodian signs the correct document for the new category.

III. Depreciation Background Service

A monthly service runs for all items where CurrentCategory == PPE:

    Calculate Salvage Value: UnitCost * Policy.StandardSalvageRate.

    Monthly Expense: (UnitCost - SalvageValue) / (UsefulLife * 12).

    Post: Add a DepreciationEntry to the ledger.

    Stop Condition: Cease posting when RemainingBookValue equals Salvage Value or if the item is reclassified to Semi-Expendable.
